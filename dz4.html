Основы работы с текстом

Заголовки и абзацы

Существуют теги для заголовков в html h1,h2,h3,h4,h5,h6, различающиеся по уровню. Тег h1 применяется для заголовка, который характеризует содержание всей страницы.

Списки

Упорядоченные списки

Упорядоченные списки отличаются тем, что каждому элементу присваивается порядковый номер. Для них используется тег ol.

Неупорядоченные списки

В неупорядоченных списках элементы отмечаются маркерами. Для них используется тег ul.

Необходимые инструменты для работы с текстом

  • Notepad++
  • VS Code
  • Sublime Text
